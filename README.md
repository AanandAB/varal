# varal

Kerala Live Varal (വരാൽ) — Farm-raised Snakehead fish ordering website.

## Features
- 3D swimming fish model (Three.js + GPU vertex shader animation)
- WhatsApp ordering with GPS location
- ₹120/piece, delivered alive
- Responsive design with scroll animations
- Red + Green Kerala-themed color palette

## Live Site
🌐 [https://aanandab.github.io/varal/](https://aanandab.github.io/varal/)

## Deploy
Just `index.html` + the `.glb` model file — pure static, no build step needed.
